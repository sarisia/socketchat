## What's this

A chat server / client using raw TCP/IP socket written in C.

## Usage

### Server

```shell
$ make server
$ ./server <ip> <port>
```

### Client

```shell
$ make client
$ ./client <ip> <port>
```

#### Commands

* `/help`
* `/exit`, `/quit`
